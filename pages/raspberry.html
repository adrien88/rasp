Réseau

Cette page est stockée sur mon raspberry pi. Pour la voir, votre application sur votre terminal, a envoyé une requête http:// qui a traversé tout le web, été redirigé par les serveurs d'OVH, puis par la box pour arriver sur le serveur web apache de mon raspberry pi. Celui ci à émit une réponse, cette page, qui elle, a fait le chemin inverse jusqu'à vous. 

 |          | ---> {           } ---> requête ---> |      | ---> |	       |
 | Terminal |      { web (OVH) }                   | BBox |      | RaspberryPi |
 |          | <--- {           } <--- réponse <--- |      | <--- |   	       |

Je chiffrerai la connexion plus tard, pour l'intant, en clair, c'est pas trop gênant.
